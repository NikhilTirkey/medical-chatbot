# 🏥 AI Medical Chatbot

An AI-powered medical chatbot built using Large Language Models and Retrieval-Augmented Generation (RAG) to provide accurate, context-based medical responses.

---

## 🚀 Features
- 💬 Intelligent medical Q&A system
- 📄 PDF-based knowledge retrieval
- 🧠 RAG (Retrieval-Augmented Generation)
- ⚡ Fast responses using Groq API
- 🌐 Flask-based web interface

---

## 🛠 Tech Stack
- Python
- LangChain
- Groq API (LLM)
- FAISS (Vector Database)
- Flask
- HTML/CSS
